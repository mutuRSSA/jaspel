📖 Buku Panduan Sistem Jaspel

SOP Penggunaan Sistem Manajemen Jasa Pelayanan RSUD - Super Lengkap & Mudah
💡 Pengantar Singkat:
Sistem ini dibuat agar Anda tidak perlu lagi mengetik manual atau mencocokkan data pasien satu per satu. Komputer yang akan mengerjakannya untuk Anda. Tugas Anda hanyalah meng-upload file Excel ke dalam sistem dengan URUTAN YANG BENAR.
🛑 ATURAN EMAS (HARAP DIBACA SEBELUM MULAI)
  1. Jangan Ubah File Excel dari SIMRS: Saat Anda mendownload laporan, biarkan file itu apa adanya. JANGAN mengganti nama kolom, menghapus baris, atau mewarnai selnya. Langsung masukkan ke sistem.
  2. Urutan Upload Tidak Boleh Terbalik: Ibarat membangun rumah, buat pondasi dulu (Laporan Kasir), baru pasang atap (Rincian Pendapatan). Kalau terbalik, rumahnya akan roboh (sistem akan error).

BAB 1: CARA MENGUNGGAH (UPLOAD) DATA BULANAN

(Lakukan ini setiap kali Anda ingin memasukkan data pasien baru ke dalam web)

Buka menu di sebelah kiri layar Anda: Klik 📝 Entri Jaspel lalu pilih 🗂️ Universal Uploader.

🛠️ LANGKAH 1: Masukkan Data KASIR (Pondasi Rumah)

Mengapa ini yang pertama? Karena file kasir berisi data identitas lengkap (Nama, No RM, dan Dokter Asli). Kita harus mendaftarkan pasiennya dulu ke sistem.

  1. Lihat tulisan "Pilih Format File yang Diupload" di layar atas.
  2. Pilih 🗂️ Laporan Kasir Rawat Jalan atau 🗂️ Laporan Kasir Rawat Inap.
  3. Tarik dan seret (drag) file Excel Kasir tersebut ke dalam kotak besar berwarna biru.
  4. Jika datanya sudah muncul di tabel bawah, klik tombol hijau "Mulai Eksekusi Database 🚀".
  5. Tunggu sampai muncul tulisan "SUKSES".
🛠️ LANGKAH 2: Masukkan Data PENDAPATAN (Pewarna Rumah)

Sekarang kita beri tahu sistem: "Pasien ini BPJS/Jasa Raharja-nya cair berapa rupiah, dan statusnya apa?"

  1. Ganti pilihan kotak di atas menjadi: 💰 Rincian Pendapatan (Multi-Sheet).
  2. Tarik dan seret file Excel Rincian Pendapatan ke dalam kotak biru.
  3. Hebatnya sistem ini: Meskipun di file Excel Anda nama dokternya hanya tertulis "UGD", sistem akan tetap mempertahankan nama dokter asli yang sudah Anda masukkan di Langkah 1!
  4. Klik tombol hijau "Mulai Eksekusi Database 🚀" dan tunggu sampai Sukses.
🛠️ LANGKAH 3: Masukkan Data USG (Pelengkap)
  1. Ganti pilihan kotak di atas menjadi: 🩻 Rekap Pemeriksaan USG.
  2. Tarik dan seret file Excel USG ke dalam kotak biru.
  3. Klik tombol hijau "Mulai Eksekusi Database 🚀". Sistem otomatis memberi tanda centang "Ya" di kolom USG pasien tersebut.

BAB 2: JIKA ADA DATA YANG SALAH (CARA EDIT MANUAL)

(Gunakan ini jika ada klaim BPJS yang tiba-tiba ditolak/direvisi, atau ingin mengubah nama dokter secara spesifik)

  1. Buka menu: 📝 Entri Jaspel lalu pilih 🏥 Database Pasien.
  2. Gunakan kotak Pencarian (ketik Nama atau No RM) untuk mencari pasien.
  3. Jika sudah ketemu, ubah datanya langsung di tabel tersebut.
PENTING TENTANG STATUS KLAIM:
✔️ Pilih "Disetujui" 👉 Jika uangnya sudah pasti turun. (Hanya status ini yang uangnya akan dihitung dan dibagikan).
❌ Pilih "Pending" / "Tidak Layak" / "VPK" 👉 Jika bermasalah. Uangnya TIDAK AKAN masuk ke kantong dokter/perawat bulan ini.

BAB 3: MENGATUR PEGAWAI & NILAI KINERJA

(Wajib dilakukan SETIAP AWAL BULAN sebelum bagi-bagi uang)

A. Karyawan Baru / Resign / Mutasi

  1. Buka menu: ⚙️ Data Master lalu pilih 👥 Karyawan & Kinerja.
  2. Tambahkan data karyawan baru atau ubah jabatan karyawan lama. Pastikan Nomor Rekening terisi benar.

B. Mengisi Nilai Kinerja Bulanan

  1. Buka menu: ⚙️ Data Master lalu pilih ⭐ Data Kinerja.
  2. Isi angka Produktivitas, Disiplin, dan Hukuman untuk setiap orang.
  3. Jika tidak ada pelanggaran, pastikan nilainya 1 (standar). Jangan biarkan 0, nanti jatah jaspel mereka hilang!

BAB 4: CARA BAGI UANG & TUTUP BUKU

(Lakukan ini saat semua data sudah benar dan uang siap ditransfer)

  1. Cek Uang: Buka 📊 Dashboard Analitik. Lihat total uang yang cair. Apakah masuk akal?
  2. Cetak Slip: Buka 📑 Laporan Jaspel. Download Slip Jaspel (PDF) untuk masing-masing orang.
  3. Kunci Data (Tutup Buku): Jika semua sudah final, Anda WAJIB menekan tombol "Arsipkan Laporan".
Apa yang terjadi saat menekan tombol "Arsipkan Laporan"?
  • Data yang sudah dibayarkan akan dihapus dari layar antrean Anda, supaya web tidak lemot.
  • Data tersebut tidak hilang! Melainkan dikunci rapat dan disimpan otomatis ke Google Drive.
  • Status uangnya berubah menjadi "Terbayarkan". Tidak ada lagi risiko uang dokter dibayar dobel bulan depan.

BAB 5: BANTUAN DARURAT (TANYA JAWAB)

Tanya: "Kok waktu saya naruh data Pendapatan, pasiennya malah numpuk di bawah bikin data baru? Padahal pasiennya udah ada!"

Jawab: Anda pasti lupa melakukan Langkah 1 (Upload Kasir). Jangan pernah upload Rincian Pendapatan sebelum meng-upload Laporan Kasir!

Tanya: "Kok saya nggak bisa login?"

Jawab: Perhatikan huruf besar/kecil (Caps Lock) pada password. Jika masih tidak bisa, minta Admin untuk mereset password Anda.

Tanya: "Saya habis copy-paste data manual dari Excel, kok dokternya masih singkatan (misal: ALVIAN)?"

Jawab: Sistem kita sangat pintar. Cukup Paste saja. Begitu diklik simpan ke database, sistem otomatis mengubah kata "ALVIAN" menjadi nama lengkap resminya.
